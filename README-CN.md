# :movie_camera: SRS_ADMIN

[English](./README.md)

一套基于 `SRS` 的流媒体系统开发环境

## :computer: 特性

* Docker-compose 集成
* 多平台支持
* 多协议支持
* 性能度量
* 链路追踪
* 反向代理

## :wrench: 构建

```shell
# 拷贝源代码至本地环境
$ git clone https://github.com/Marksagittarius/srs_admin

$ cd srs_admin

# 部署容器集群
# 保证您的电脑已经部署了 docker 与 docker-compose 开发环境
$ docker-compose -d
```
